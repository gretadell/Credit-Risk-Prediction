# Credit Risk Prediction

The goal of the project is to estimate the likelihood of an individual experiencing financial distress within the next two years.

The project includes data exploration, missing value imputation, outlier handling, feature engineering, and feature selection.

Several classification models were tested, including Logistic Regression, Bagging, XGBoost, and Histogram-based Gradient Boosting.

The final model selection focused on balancing recall, precision, ROC AUC, and overall predictive performance.

Particular attention was given to the challenges of imbalanced data and financial risk classification.

The analysis was developed in Python using a Jupyter Notebook.
